一、创建Django项目

二、在终端中创建应用 python manage.py startapp 应用名

三、在项目包中的settings.py文件中添加应用 INSTALLED_APPS = [
'django.contrib.admin',
'django.contrib.auth',
'django.contrib.contenttypes',
'django.contrib.sessions',
'django.contrib.messages',
'django.contrib.staticfiles',
'stu' ]

四、确定访问路径 http://127.0.0.1:8000/student/

五、配置项目跟路由
from django.conf.urls.import url,include
from django.contrib import admin
from django.urls import path
from PythonWEB import views
urlpatterns = [
url(r'^student/', include('stu.urls'))
]

六、在应用包下创建并配置urls.py文件
from django.conf.urls import url
from . import views
urlpatterns = [
url(r'^$', views.login_view)
]

七、在应用包下views.py文件中创建处理函数
from django.shortcuts import render
def login_view(request):
return render(request, 'login.html')
